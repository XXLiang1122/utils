# video2sketch

## [click me](https://xxliang1122.github.io/video2sketch/dist/)   

### step 1
```bash
git clone https://github.com/XXLiang1122/video2sketch.git
```

### step 2
```bash
npm install
```

### step 3
```bash
npm run dev
```
